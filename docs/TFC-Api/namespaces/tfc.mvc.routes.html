RouteRegex¶

RouteRegex class file 正则路由

一.正则路由例子： URL：http://domain.com/archive/2012

$route = new RouteRegex(
    'archive/(\d+)',
    array(
        'controller' => 'archive',
        'action'     => 'show'
    )
);
$value = array(
    'controller' => 'archive',
    'action'     => 'show',
    1            => '2012'
);

二.上面例子中整数键不容易管理，下面方式解决：

$route = new RouteRegex(
    'archive/(\d+)',
    array(
        'controller' => 'archive',
        'action'     => 'show'
    ),
    array(
        1 => 'year'
    )
);
$value = array(
    'controller' => 'archive',
    'action'     => 'show',
    'year'       => '2012'
);

三.如果URL的地址是：http://domain.com/archive，即后面缺省2012时，下面方式解决：

$route = new RouteRegex(
    'archive(?:/(\d+))?',
    array(
        1            => '2012',
        'controller' => 'archive',
        'action'     => 'show'
    ),
    array(
        1 => 'year'
    )
);
$value = array(
    'controller' => 'archive',
    'action'     => 'show',
    'year'       => '2012'
);

四.URL中有多个匹配： URL：http://domain.com/iphper/page/8

$route = new RouteRegex(
    '(\w+)/page/(\d+)',
    array(
        'controller' => 'profile',
        'action'     => 'userinfo'
    ),
    array(
        2 => 'page'
    )
);
$value = array(
    'controller' => 'profile',
    'action'     => 'userinfo',
    1            => 'iphper',
    'page'       => '8'
);

五.URL中有多个参数： URL：http://domain.com/archive/2012/username/iphper/page/2

$route = new RouteRegex(
    'archive/(\d+)',
    array(
        'controller' => 'archive',
        'action'     => 'show'
    ),
    array(
        2 => 'page'
    )
);
$value = array(
    'controller' => 'archive',
    'action'     => 'show',
    1            => '2012',
    'username'   => 'iphper',
    'page'       => '2'
);

RouteRewrite¶

RouteRewrite class file 标准路由

一.标准路由例子： URL：http://domain.com/author/iphper

$route = new RouteRewrite(
    'author/:username',
    array(
        'controller' => 'profile',
        'action'     => 'userinfo'
    )
);
$value = array(
    'controller' => 'profile',
    'action'     => 'userinfo',
    'username'   => 'iphper'
);

二.规则匹配： URL：http://domain.com/archive/2012

$route = new RouteRewrite(
    'archive/:year',
    array(
        'controller' => 'archive',
        'action'     => 'show'
    ),
    array(
        'year'    => '\d+'
    )
);
$value = array(
    'controller' => 'archive',
    'action'     => 'show',
    'year'       => '2012'
);
如果URL：http://domain.com/archive/test，将不匹配，并且尝试匹配下一个路由

三.有缺省值时，规则匹配： URL：http://domain.com/archive

$route = new RouteRewrite(
    'archive/:year',
    array(
        'year'       => '2012',    // 缺省值
        'controller' => 'archive',
        'action'     => 'show'
    )
);
$value = array(
    'year'       => '2012',
    'controller' => 'archive',
    'action'     => 'show'
);

四.模仿模块路由： URL：http://domain.com/passport/profile/userinfo/username/iphper

$route = new RouteRewrite(
    ':module/:controller/:action/*',
    array(
       'module' => 'default'    // 缺省值
    )
);
$value = array(
    'module'     => 'passport',
    'controller' => 'profile',
    'action'     => 'userinfo',
    'username'   => 'iphper'
);
如果URL：http://domain.com/archive/test，将不匹配，并且尝试匹配下一个路由
