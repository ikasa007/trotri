RouteRegex

Extends \tfc\mvc\routes\Route

RouteRegex class file 正则路由

一.正则路由例子： URL：http://domain.com/archive/2012

$route = new RouteRegex(
    'archive/(\d+)',
    array(
        'controller' => 'archive',
        'action'     => 'show'
    )
);
$value = array(
    'controller' => 'archive',
    'action'     => 'show',
    1            => '2012'
);

二.上面例子中整数键不容易管理，下面方式解决：

$route = new RouteRegex(
    'archive/(\d+)',
    array(
        'controller' => 'archive',
        'action'     => 'show'
    ),
    array(
        1 => 'year'
    )
);
$value = array(
    'controller' => 'archive',
    'action'     => 'show',
    'year'       => '2012'
);

三.如果URL的地址是：http://domain.com/archive，即后面缺省2012时，下面方式解决：

$route = new RouteRegex(
    'archive(?:/(\d+))?',
    array(
        1            => '2012',
        'controller' => 'archive',
        'action'     => 'show'
    ),
    array(
        1 => 'year'
    )
);
$value = array(
    'controller' => 'archive',
    'action'     => 'show',
    'year'       => '2012'
);

四.URL中有多个匹配： URL：http://domain.com/iphper/page/8

$route = new RouteRegex(
    '(\w+)/page/(\d+)',
    array(
        'controller' => 'profile',
        'action'     => 'userinfo'
    ),
    array(
        2 => 'page'
    )
);
$value = array(
    'controller' => 'profile',
    'action'     => 'userinfo',
    1            => 'iphper',
    'page'       => '8'
);

五.URL中有多个参数： URL：http://domain.com/archive/2012/username/iphper/page/2

$route = new RouteRegex(
    'archive/(\d+)',
    array(
        'controller' => 'archive',
        'action'     => 'show'
    ),
    array(
        2 => 'page'
    )
);
$value = array(
    'controller' => 'archive',
    'action'     => 'show',
    1            => '2012',
    'username'   => 'iphper',
    'page'       => '2'
);
