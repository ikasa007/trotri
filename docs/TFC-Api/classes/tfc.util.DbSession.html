DbSession

Implements \tfc\ap\interfaces\SessionSaveHandler

DbSession class file 用MySQL数据库存储会话，默认自动创建SESSION表 手动建表时，session_id一定要加主键约束，不然SQL:REPLACE语句会添加很多新记录
