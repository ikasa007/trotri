Cookie¶

Cookie class file Cookie管理类，所有的Cookie都应该加密后保存

配置 /cfg/app/appname/main.php： <pre> return array ( 'cookie' => array ( 'key_name' => 'authentication', // 密钥配置名 'domain' => '.trotri.com', // Cookie的有效域名，缺省：当前域名 'path' => '/', // Cookie的有效服务器路径，缺省：/ 'secure' => false, // FALSE：HTTP和HTTPS协议都可传输；TRUE：只通过加密的HTTPS协议传输，缺省：FALSE 'httponly' => true, // TRUE：只能通过HTTP协议访问；FALSE：HTTP协议和脚本语言都可访问，容易造成XSS攻击，缺省：TRUE ), ); </pre>
